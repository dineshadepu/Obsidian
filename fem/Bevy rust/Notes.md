
# Boiler plate code
#bevy/essential_code
Every code needs these following lines to start with in Bevy.
```rust
fn main(){
	App::new()
		.add_plugins(DefaultPlugins)
		.run();
}
```

This will add lot of plugins, for example, we will now have a basic window opened up while running this example.
